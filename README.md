# AI Research Digest Agent

An autonomous AI agent system that automatically collects, summarizes and organizes AI industry updates into daily research reports.

## Features

- AI news collection
- GitHub Trending analysis
- Automated summarization
- Multi-agent workflow
- Long-chain reasoning
- Markdown report generation
- GitHub Actions automation

## Workflow

1. Collect AI news
2. Filter relevant content
3. Generate summaries
4. Analyze trends
5. Export reports automatically

## Tech Stack

- Python
- OpenAI API
- GitHub Actions

## Example Output

Daily AI research reports generated automatically.
